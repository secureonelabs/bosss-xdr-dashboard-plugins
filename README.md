## Welcome

This repository contains the plugins for [BOSSS XDR Dashboard](https://github.com/secureonelabs/bosss-xdr-dashboard).

BOSSS XDR is a security detection, visibility, and compliance open source project. It helps you to gain deeper security visibility into your infrastructure by monitoring hosts at an operating system and application level.
